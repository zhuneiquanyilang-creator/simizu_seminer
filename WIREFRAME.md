ポイント:
- **「次回の発表」** を最上部に大きく表示 → 「次は誰の担当か」がすぐ分かる

- 行クリックで③の節詳細（レジュメ）へ

> ❓検討: 担当者の割り当て・発表日の編集を「②の一覧で直接」やるか、
> 「③詳細画面で」やるか。→ 一覧でインライン編集できると楽（推奨）。

│                                              [ 保存 ]        │

│  │  │     または [ ファイルを選択 ]   ※PDFのみ      │     │ │
│  │  └─────────────────────────────────────────────┘     │ │
│  │     投稿者名: [ 田中            ]   [ アップロード ]   │ │

│  ┌── 補足メモ（任意・後回し可）─────────────────────────┐ │

ポイント:
- 上部で **担当者・発表日・状態を編集** → 保存で②一覧にも反映

- **アップロード枠**: ドラッグ&ドロップ or 選択。投稿者名は手入力（共有PWのため）
- **補足メモ** は枠だけ用意し、MVPでは「後で実装」表示（後回し方針どおり）

## 確認したいポイント

**確定済み:**
- [x] サイト表示名 … **清水研究室**
- [x] 状態区分 … **担当未決定 / 準備中 / 未完了 / 発表済み**（4区分）

**状態の定義（確定）:**
- ⚪ 担当未決定 … 担当者未割り当て
- 🔵 準備中 … 担当が決まりレジュメ準備中
- 🟡 未完了 … 発表したが時間切れで来週へ持ち越し
- ✅ 発表済み … 発表完了
